========================================
QUICK START GUIDE
========================================

3 SIMPLE STEPS TO GET STARTED:

1. IMPORT DATABASE
   - Open phpMyAdmin (http://localhost/phpmyadmin)
   - Click "Import"
   - Choose "database_schema.sql"
   - Click "Go"

2. CONFIGURE
   - Open config/db.php
   - Set your MySQL password if needed
   - Save file

3. LOGIN
   - Open: http://localhost/your-folder/login.php
   - Instructor: john.smith@university.edu / password123
   - Student: emma.wilson@student.edu / password123
